# Renewable-energy-analytics-dashboard-Tableau-
Interactive data visualisation project showcasing Tableau dashboard design, data modelling, and analytical insight.

**Project Summary**

This project demonstrates my ability to design interactive Tableau dashboards that support real-world analytical questions. The dashboard explores global renewable energy production across countries, energy sources, and time, enabling users to identify trends, compare regions, and drill into specific technologies.

The focus of this repository is Tableau dashboard design, interactivity, and insight delivery.
